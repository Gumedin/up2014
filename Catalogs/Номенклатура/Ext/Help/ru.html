Справочник "Номенклатура"

Справочник "Номенклатура" предназначен для хранения информации о номенклатурных позициях двух типов товар и услуга. Позиции номенклатуры можно дополнительно классифицировать по видам номенклатуры.

Вид номенклатуры – это дополнительная классификация товаров, более обобщенная, чем тип товара: товар или услуга. В частном случае вид номенклатуры может совпадать с указанными типами номенклатуры. Однако для более детальной классификации можно использовать несколько видов номенклатуры, имеющих один и тот же тип.

Пример. Если необходимо разделять услуги, которые оказывает наше предприятие, и услуги, оказываемые сторонними организациями, то можно создать два вида номенклатуры: внутренние услуги и сторонние услуги с одним и тем же типом – услуга.

Вид номенклатуры также применяется при ведении учета товаров в разрезе характеристик (цвет, размер, объем морозильной камеры, габариты и т. д.). Для вида номенклатуры можно назначить определенный набор характеристик, который будет общим для всех позиций номенклатуры данного вида. Аналогичным образом можно назначить список общих упаковок для вида номенклатуры.

При описании номенклатурных позиций необходимо определить ряд дополнительных свойств: артикул, внутрифирменное наименование, публичное наименование, изображения.

Кроме того, для позиций номенклатуры определяются ценовая группа, серийные номера и т.д. Для каждой номенклатурной позиции могут быть заданы соответствия позиций номенклатуры позициям номенклатуры партнеров, отличающиеся, например, артикулом или наименованием. Для импортных товаров устанавливается признак ведения учета по ГТД и указывается тот номер ГТД и страна происхождения, которая будет автоматически заполняться в документах поставки импортных товаров.

Указывается вид, к которому следует отнести данную позицию номенклатуры. (Виды номенклатуры)

Группа доступа

Единица хранения

В этой единице измерения ведется учет складских остатков товаров и указывается количество номенклатуры в одной упаковке. (Единицы измерения)

Использовать серийные номера

Признак того, что для элемента номенклатуры можно задать серийные номера. (Булево)

Набор упаковок

Набор упаковок для данной номенлатуры. Если указано значение "Индивидуальный набор упаковок", то упаковки ведутся отдельно для этой позиции номенклатуры, если значение не заполнено, то упаковки для этой позиции не используются. (Наборы упаковок)

Наименование для печати

Наименование номенклатуры, которое будет печататься во всех документах. (Строка)

Складская группа

Классификация номенклатуры по признакам складской обработки. В разрезе складских групп номенклатуры задаются правила автоматического размещения номенклатуры на складе с адресным хранением остатков. (Складские группы номенклатуры)

Ставка НДС

Определяется ставка НДС товара или услуги (Перечисление)

Тип номенклатуры

Содержит тип, к которому относится номенклатура. Данный реквизит копируется из вида номенклатуры. В элементе номенклатуры реквизит не может быть изменен пользователем. (Товар/Услуга)
